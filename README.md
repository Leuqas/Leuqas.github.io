# Leuqas Yabot Portfolio

Single-page portfolio website built with React, TypeScript, Tailwind CSS, and shadcn-compatible structure.

## Stack

- React + TypeScript (Vite)
- Tailwind CSS
- shadcn-compatible structure (`src/components/ui`)
- GSAP (`ScrollTrigger`)
- lucide-react

## Run

1. Install dependencies:

```bash
npm install
```

2. Start development server:

```bash
npm run dev
```

3. Build production bundle:

```bash
npm run build
```
